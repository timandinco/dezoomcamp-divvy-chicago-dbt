# Data Engineering Zoomcamp - Explore Chicago Divvy
A pipeline to investigate Divvy transportation in Chicago


This is my project to help me cement some of the course material in Data Engineering Zoomcamp. After following along in the course material, nothing helps to cement that knowledge better than to struggle through a project of my own choosing. For that, I've selected the data from Divvy bikes in the Chicago area (https://divvybikes.com/). The data is similar to the NY Taxi Trip data used during the class.
